# Buzz

Opinionated modern event based application development framework. 

## Features

- UIEvents
- Commands
  - Navigation
- AppEvents

## Installation

```yaml
buzz:
    git:
      url: git@github.com:LuisBurgos/buzz.git
      ref: main
```

## Usage

### Initialization

````dart
Buzz..init(
    navigator: MyAppNavigator(),
),
````
